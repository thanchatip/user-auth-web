# User Authentication Web

User Authentication Web using nuxt3

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on http://localhost:3001

```bash
npm run dev
```
